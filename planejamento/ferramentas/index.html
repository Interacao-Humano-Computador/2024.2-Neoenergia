Ferramentas

Introdução

Para a realização deste projeto, foram utilizadas algumas ferramentas que auxiliam nas tarefas, documentação e comunicação feitas pelos membros da equipe. As ferramentas utilizadas são apresentadas na Tabela 1.


Tabela 1 - Ferramentas Utilizadas

Ferramenta Finalidade
Figma Criação de artefatos gráficos.
Github Repositório para organização do projeto.
Google Docs Criação e edição de arquivos.
Google Planilhas Criação da planilha de grade de horários.
Microsoft Teams Realização e gravação de reuniões.
MkDocs Criação da página de documentação.
This Person Does Not Exist Criação de imagens das personas.
Visual Studio Code Edição dos arquivos da git page.
WhatsApp Principal plataforma de comunicação rápida entre a equipe.
YouTube Publicação das apresentações gravadas.
